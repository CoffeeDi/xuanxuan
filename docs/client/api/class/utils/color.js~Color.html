    public contrast(dark: String, light: String, threshold: number): Color source

    根据当前颜色亮度明暗程度返回一个对比色

    Params:

    NameTypeAttributeDescription
    dark String
    • nullable: true

    如果当前颜色为浅色，则返回此值指定的深色作为对比色，如果不指定则使用纯黑色
    light String
    • nullable: true

    如果当前颜色为深色，则返回此值指定的浅色作为对比色，如果不指定则使用纯白色
    threshold number
    • optional
    • default: 0.43

    判断是否为深色的阈值，可选范围 0～1

    Return:

    Color
